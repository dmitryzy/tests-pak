Здесь рассмотрены примеры использования макрокоманд рассматриваемых в статьях.
Пример включает следующие файлы и каталоги:
/primer/
--template.tex Шаблон (только для чтения), содержащий список основных макрокоманд
--test_variant.tex Пример реализации шаблона *.tex
--test_variant.pdf Пример *.pdf
--readme.txt 
--/pakage/
Каталог содержит файлы с макрокомандами
----texarray.tex Макросы для работы с массивами
----testsmakr.tex Макросы для создания тестовых заданий на основе одной группы вопросов
----testsmakr2.tex Макросы для создания тестовых заданий из нескольких групп вопросов (окончательная версия)
--/kontrol/
Каталог содержит файлы с вопросами
----r1.tex
----r2.tex
----r3.tex
----r4.tex
