Перечень основных команд
	После подробного описания приведем краткий перечень основных команд, определенных в файле testsmakr2.tex и необходимых для работы. они разделены на 6 групп.
1) Параметры.
\trueotv Маркер правильных ответов.
\printsimple печатать без заголовков.
\printall Печатать с заголовками.
\setprinttrueotv Команда помечать правильные ответы при печати.
\setnoprinttrueotv Команда не помечать правильные ответы при печати.
\ResetVariant{<Номер варианта>} Установка заданного значения номера варианта.

2) Команды разметки.
\Qvery{вопрос}\endQvery Команда для обозначения вопроса.
\VarQveryOtv Команда для обозначения варианта ответа.
\AddNewSectionFile{<Имя файла>} Добавление в группу вопросов из файла.
\AddNewSection Объявление новой группы вопросов.

3) Команды вывода вопросов
\printQSection{<Номер группы>}{<Номер вопроса в группе>} Напечатать определенный вопрос из заданной группы вопросов.
\printQ{<Номер вопроса в общем списке>} Команда, позволяющая напечатать вопрос с заданным номером из общего списка вопросов.

4) Команды печати списков вопросов и вариантов.
\makelistqvery Команда для вывода списка всех вопросов из всех групп по порядку.
\printbilet{<Номера вопросов>} Напечатать билет с указанными номерами вопросов. Номера вопросов разделяются командой \and.
\twotestblok{<Номера вопросов 1>}{<Номера вопросов 2>} Напечатать блок 2 билета на листе.
\fiertestblok{<Номера вопросов 1>}{<Номера вопросов 2>}{<Номера вопросов 3>}{<Номера вопросов 4>} Напечатать блок 4 билета на листе.
\printkontrol{<Кол-во вариантов>}{<Параметр 1>}{<Параметр 2>} Распечатать комплект уникальных вариантов.
\printvariant{<Номер варианта>}{<Параметр 1>}{<Параметр 2>} Распечатать заданный вариант.
Параметр 1 задает номер начальной задачи для 1-го раздела для 1-го варианта.
Параметр 2 задает смещение между вопросами варианта.
\printsection{<Номер группы>}{<Заголовок 1>}{<Заголовок 2>} Распечатать группу вопросов.

5) Переменные шаблона
\kafedra{<Название кафедры>}
\zavkaf{<Фамилия И.О.>}
\datautv{<чч.мv.гг>}
\vuz{<Название ВУЗа>}
\bilettitle{<Заголовок билета>}
\biletauthor{<Фамилия И.О.>}
\predmet{<Название предмета>}
\spez{<Специальности>}

6) Команды и окружения.
\ControlHeader Команда для заголовка контрольной.
\BiletHeader Команда для заголовка билета.
\BiletFooter Команда для нижней части билета.
EkzBilet Окружение для билета
ListQvery Окружение для списка вопросов
